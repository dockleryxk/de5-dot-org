2. Set viewport to 375px width (iPhone SE)
3. Observe navbar and content layout
4. Click hamburger menu icon
5. Resize to 768px (tablet), then 1024px (desktop)

**Expected:**

**Why human:** Responsive behavior requires viewport manipulation and visual verification of layout shifts across breakpoints.

#### 4. Self-Hosted Inter Font

**Expected:** 
- Font requests go to `/_fonts/` path (local server)
